# Дашборд блокировок

Визуализация времени и причин блокировок по задачам очередей POOLING, DOSTAVKAPIKO, UDOSTAVKA.

## Переменные окружения

| Переменная    | Описание                        |
|---------------|---------------------------------|
| TRACKER_TOKEN | OAuth-токен Яндекс Трекера      |
| ORG_ID        | ID организации (по умолчанию 7405124) |
| TURSO_URL     | URL базы данных Turso           |
| TURSO_TOKEN   | Токен Turso                     |
